# ataka-help-backend
This is a sample Attacka Help api server

## How to run
Install docker [docker](https://docs.docker.com/engine/install/).
Install docker-compose [docker-compose](https://pkg.go.dev/github.com/docker/compose/v2#section-readme)

Run command
```
make run
```

## How to stop
Run command
```
make stop
```

## How to rebuild
Run command
```
make rebuild
```
